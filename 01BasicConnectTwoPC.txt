Steps to ping from PC-A to PC-B
===============================
1. Configure Ip address and subnet mask to PCA-A 
    Ip Address: 192.168.0.1
    Subnet Mask: 255.255.0.0
2. Configure Ip address and subnet mask to PCA-A 
    Ip Address: 192.168.0.2
    Subnet Mask: 255.255.0.0
3. Open Command Prompt of PC-A. Then write ping 172.168.0.2
N.B: This command will send 4 ping message to PC-B.
